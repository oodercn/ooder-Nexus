# 技能同步模块使用说明

## 概述

技能同步模块允许您在多台设备之间同步技能，或将技能备份到云端。适合个人多设备使用、家庭分享、小团队协作等场景。

## 访问方式

打开浏览器，访问：
```
http://localhost:8081/console/pages/skillcenter-sync/sync-dashboard.html
```

## 功能介绍

### 1. 同步概览

进入数据同步页面，您可以看到：

#### 统计卡片
- **总任务**：显示所有同步任务数量
- **已完成**：显示已完成的同步任务
- **进行中**：显示正在执行的同步任务
- **已同步技能**：显示已同步的技能数量
- **成功率**：显示同步任务的成功率

#### 标签页
页面包含三个标签页：
1. **同步任务**：管理同步任务
2. **技能管理**：查看和管理可同步的技能
3. **同步历史**：查看同步历史记录

### 2. 同步任务管理

#### 创建同步任务

1. 点击 **"新建同步"** 按钮
2. 填写任务信息：
   - **任务名称**：给任务起个名字（如"备份到云端"）
   - **同步类型**：
     - **上传到云端**：将本地技能备份到云端
     - **下载到本地**：从云端恢复技能到本地
     - **双向同步**：合并本地和云端的技能
3. 点击 **"创建"** 按钮

#### 执行同步任务

1. 在任务列表中找到待执行的任务
2. 点击 **"执行"** 按钮
3. 等待任务完成，可以查看进度条

#### 取消同步任务

1. 在任务列表中找到正在执行的任务
2. 点击 **"取消"** 按钮
3. 任务状态将变为"已取消"

#### 删除同步任务

1. 在任务列表中找到要删除的任务
2. 点击 **"删除"** 按钮
3. 确认删除后，任务将被移除

#### 过滤任务

使用状态过滤器可以查看特定状态的任务：
- **全部状态**：显示所有任务
- **待执行**：显示等待执行的任务
- **进行中**：显示正在执行的任务
- **已完成**：显示已完成的任务
- **失败**：显示执行失败的任务

### 3. 技能管理

#### 查看可同步技能

切换到 **"技能管理"** 标签页，您可以看到：
- 所有可同步的技能列表
- 每个技能的名称、版本、分类、大小
- 技能是否已同步的状态

#### 批量同步技能

1. 在技能管理页面，勾选要同步的技能
2. 点击 **"批量同步"** 按钮
3. 选择同步类型（上传/下载）
4. 系统将自动创建同步任务并执行

#### 上传单个技能

1. 找到要上传的技能
2. 点击技能卡片上的 **"上传"** 按钮
3. 确认上传后，系统将创建上传任务

#### 下载单个技能

1. 找到要下载的技能
2. 点击技能卡片上的 **"下载"** 按钮
3. 确认下载后，系统将创建下载任务

### 4. 查看同步历史

切换到 **"同步历史"** 标签页，您可以查看：
- 所有同步操作的历史记录
- 每次同步的时间、方向、状态
- 同步的技能列表

## 使用场景

### 场景1：个人多设备同步

**需求**：您在办公室电脑和家庭电脑上都使用 ooderNexus，希望技能保持一致。

**操作步骤**：
1. 在办公室电脑上：
   - 创建 **"上传到云端"** 任务
   - 执行同步，将所有技能备份到云端

2. 在家庭电脑上：
   - 创建 **"下载到本地"** 任务
   - 执行同步，从云端恢复技能

3. 定期同步：
   - 每周执行一次双向同步
   - 保持两台电脑的技能一致

### 场景2：技能备份

**需求**：您想备份重要的技能，防止数据丢失。

**操作步骤**：
1. 选择 **"上传到云端"** 类型
2. 选择要备份的技能
3. 执行任务
4. 备份完成后，可以在云端查看已备份的技能

### 场景3：团队技能分享

**需求**：您创建了一个好用的技能，想分享给团队成员。

**操作步骤**：
1. 上传技能到云端
2. 告诉团队成员技能ID
3. 团队成员通过 **"下载到本地"** 获取技能

## 注意事项

### 1. 同步冲突

当本地和云端都有同一技能的不同版本时，可能会产生冲突。

**解决方法**：
- 系统会保留两个版本
- 您可以手动选择保留哪个版本
- 或者合并两个版本的配置

### 2. 网络要求

- 同步需要网络连接
- 建议在稳定的网络环境下执行同步
- 大文件同步可能需要较长时间

### 3. 存储限制

- 单任务最多同步100个技能
- 最大并发任务数为3
- 任务记录保留30天

### 4. 数据安全

- 同步数据默认加密传输
- 建议定期备份重要技能
- 敏感技能请谨慎分享

## 常见问题

### Q1: 同步任务一直显示"进行中"怎么办？

**A**: 
1. 检查网络连接是否正常
2. 等待一段时间，可能是技能文件较大
3. 如果长时间无响应，可以取消任务后重试

### Q2: 同步失败怎么办？

**A**:
1. 查看错误信息，了解失败原因
2. 检查网络连接
3. 确认目标位置可用（云端存储空间是否充足）
4. 重试同步任务

### Q3: 如何查看同步进度？

**A**:
- 在任务列表中，可以看到进度条
- 点击任务可以查看详细信息
- 包括已处理项目数和总项目数

### Q4: 可以自动同步吗？

**A**:
- 当前版本需要手动执行同步
- 可以在系统设置中配置定时同步（未来版本支持）

### Q5: 同步会覆盖本地技能吗？

**A**:
- **上传到云端**：不会覆盖本地技能
- **下载到本地**：如果本地存在同名技能，会提示冲突
- **双向同步**：会尝试合并，冲突时保留两个版本

## 故障排除

### 无法创建同步任务

**可能原因**：
- 网络连接问题
- 服务器繁忙

**解决方法**：
1. 检查网络连接
2. 刷新页面后重试
3. 查看系统日志

### 技能列表为空

**可能原因**：
- 还没有安装任何技能
- 技能数据加载失败

**解决方法**：
1. 确认已安装技能
2. 刷新页面
3. 检查浏览器控制台是否有错误

### 页面显示异常

**可能原因**：
- 浏览器缓存问题
- CSS样式加载失败

**解决方法**：
1. 清除浏览器缓存
2. 按 Ctrl+F5 强制刷新
3. 检查网络连接
